## Editor
### Layout
- ^b: toggle sidebar


### Side By Side
- ^\: window split
- ^1, ^2: window navigation

### Explorer

### Search
- ^+F: search content
- ^+J: search advanced

### Command Palette
- ^p: search file name
- ^tab: cycle open files
- F1: editor commands
- ^+O: navigate to a specific symbol in a file
- ^G: goto line

## Languages
### Markdown
- ^+V : toggle between the code and a preview mode
- ^<SP>: intellisense
- fenced code blocks: ```
